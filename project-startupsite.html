StartupSite

A modern SaaS landing page with lead capture, analytics, and A/B testing capabilities.

StartupSite is a high-converting landing page built for a B2B SaaS startup preparing for their public launch. The page was designed with conversion optimization principles and includes a lead capture system with email automation integration.

The page features a hero section with animated illustrations, feature highlights with interactive demos, pricing tables, customer testimonials, and a sticky CTA. I implemented A/B testing using a custom solution that routes users to different variants and tracks conversion metrics.

The lead capture system integrates with Mailchimp for email sequences and includes real-time validation, spam protection, and a dashboard for monitoring signup metrics. The page achieved a 28% conversion rate in the first month of launch.

Tech Stack

Next.js, Framer Motion, Mailchimp API, Vercel, Mixpanel

Key Features

Lead capture, A/B testing, Email automation, Analytics dashboard, Animated UI
