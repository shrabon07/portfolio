PulseFit

A comprehensive fitness tracking app with personalized workout plans, progress analytics, and social challenges.

Launch Live Demo

PulseFit is a cross-platform fitness tracking application designed to help users reach their health goals through data-driven insights. It offers personalized workout recommendations powered by user data and progress history.

Built with React Native for a native feel on both iOS and Android, PulseFit integrates with HealthKit API for seamless health data synchronization. Firebase powers real-time social features and cloud storage.

Key features include AI-generated workout plans customized to fitness levels and equipment, beautiful progress charts showing strength and cardio improvements, an exercise library with HD video guides, and social challenges with badges.

Tech Stack

React Native, Firebase, Chart.js, HealthKit API, Node.js

Key Features

AI workout plans, Progress visualizations, 500+ exercise library, Social challenges & badges
