UrbanCart

A multi-vendor marketplace platform connecting local sellers with customers through a unified shopping experience.

UrbanCart is a multi-vendor marketplace that enables local businesses to set up online storefronts quickly. Each vendor gets their own branded store within the platform, with individual product catalogs, pricing, and shipping configurations.

The platform handles complex marketplace logic including commission-based revenue sharing, vendor payout automation, and a centralized shopping cart that aggregates items from multiple vendors into a single checkout flow. The real-time messaging system allows buyers and vendors to communicate directly.

I built a vendor dashboard with analytics, order management, and product CRUD operations. The admin panel provides oversight across all vendors with dispute resolution tools and platform-wide promotion management. UrbanCart onboarded 50+ vendors in its first quarter.

Tech Stack

React, Node.js, MongoDB, Stripe Connect, AWS S3

Key Features

Multi-vendor support, Commission engine, Vendor dashboard, Real-time messaging, Payout automation
