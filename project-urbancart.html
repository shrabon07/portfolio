UrbanCart

A powerful multi-vendor marketplace platform that enables independent sellers to reach millions of customers.

Launch Live Demo

UrbanCart is a comprehensive multi-vendor marketplace solution that empowers sellers to create their own storefronts within a larger marketplace ecosystem. Each vendor gets their own branded store while benefiting from shared traffic.

Built with a microservices architecture using Next.js and Express.js with MongoDB, UrbanCart handles vendor onboarding, product management, payment splitting via Stripe Connect, and multi-carrier shipping integrations.

Key features include customizable vendor storefronts with dedicated URLs, a real-time messaging system for buyer-seller communication, end-to-end order tracking with automated status updates, and a verified purchase review system.

Tech Stack

Next.js, Express.js, MongoDB, Stripe Connect, Socket.io, Redis

Key Features

Multi-vendor storefronts, Real-time buyer chat, Order tracking, Verified review system
